## 核心特性

### V1 核心特性
- **aToken 机制**：实时累积利息的代币
- **双利率模式**：稳定利率和浮动利率借贷
- **闪电贷功能**：无需抵押的即时借贷
- **清算机制**：自动化的风险管理

### V2 新增特性
- **债务代币分离**：稳定债务和浮动债务独立管理
- **激励系统**：AAVE 代币激励机制
- **抵押品管理优化**：独立的抵押品管理合约
- **多链支持**：支持多个区块链网络
- **Gas 优化**：显著降低交易成本

### V3 革命性特性
- **效率模式（eMode）**：为相关资产提供更高的资本效率
- **隔离模式（Isolation Mode）**：支持高风险资产的隔离借贷
- **跨链桥接**：支持跨链资产的无抵押铸造
- **供应和借贷上限**：更精细的风险管理
- **债务上限**：隔离模式的债务限制机制
- **增强的清算**：更灵活的清算策略
